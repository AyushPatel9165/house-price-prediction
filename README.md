# House Price Prediction

## Overview
This is a Machine Learning project that predicts house prices using Linear Regression. The project includes data preprocessing, model training, evaluation, and deployment using Flask web application.

## Problem Statement
The goal of this project is to predict the price of a house based on input features such as area, number of bedrooms, location, etc.

## Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Model training using Linear Regression
- Model evaluation
- Flask web application for prediction

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Flask
- HTML/CSS

- ##  Run
- pip install -r requirements.txt
- python app.py
